description
you have obtained a file from a pretty small and unheard of video game. Can you figure out what it is? Can you find out how to access the flag hidden within?

files to give participants
level.dat

guide
Load the minecraft save file into an NBT analysis tool and navigate to the players inventory. One item contains the flag.
https://irath96.github.io/webNBT/

If you own minecraft, you can create a new world and then replace the level.dat file with the one supplied by this activity. You can then load the level and your inventory will have an item containing the flag.
